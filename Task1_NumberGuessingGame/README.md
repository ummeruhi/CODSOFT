# 🎯 Ultimate Number Guessing Game (Java GUI)

An interactive **Number Guessing Game built using Java Swing GUI** that challenges players to guess a randomly generated number within limited attempts. The game provides hints, scoring, leaderboard, and multiple difficulty levels.

This project demonstrates **Java GUI development, event-driven programming, and game logic implementation**.

---

## 🚀 Features

- 🎮 Interactive **Java Swing GUI**
- 👤 Player name input
- 🎯 Multiple difficulty levels
  - Easy (1–50)
  - Medium (1–100)
  - Hard (1–500)
  - Custom range mode
- 🔢 Random number generation
- 💡 Smart hint system
- 🔥 Hot / Warm / Cold feedback
- 📊 Score tracking
- 🏆 Best score tracking
- 📈 Rounds won counter
- 📋 Session leaderboard
- ⚠ Input validation
- 🎨 Modern dark-themed GUI interface

## 🎮 How to Play

1. Enter your **player name**
2. Select a **difficulty level**
3. Click **Start Game**
4. Enter your guess in the input field
5. Use hints and feedback to guess the correct number
6. Try to guess using **minimum attempts to get higher score**

---

## 📊 Scoring System

Score is calculated based on remaining attempts.

```
Round Score = Remaining Attempts × 10
```

Fewer attempts result in higher scores.
