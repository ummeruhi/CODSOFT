# 💱 Simple Currency Converter (Java)

A lightweight **Currency Converter Desktop Application** built using **Java Swing**.
The program allows users to convert an amount from one currency to another using **live exchange rates fetched from an online API**.

---

## 🚀 Features

* Convert currency between multiple countries
* Select **base currency** and **target currency**
* Enter an amount to convert
* Fetch **real-time exchange rates from API**
* Simple **Java Swing GUI**
* Instant conversion result display
* Easy-to-understand code (perfect for beginners)

---

## 🖥️ Application Interface

The application contains:

* Amount input field
* Base currency selection
* Target currency selection
* Convert button
* Converted amount display

---

## ▶️ How to Run the Project

### 1️⃣ Compile the program

```
javac SimpleCurrencyConverter.java
```

### 2️⃣ Run the application

```
java SimpleCurrencyConverter
```

After running, a **Currency Converter GUI window** will open.

## 🌐 API Used

This project uses the **Exchange Rate API** to fetch live currency rates.

Example endpoint:

```
https://open.er-api.com/v6/latest/USD
```
