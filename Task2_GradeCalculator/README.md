# 🎓 Advanced Grade Calculator (Java GUI)

A **premium Java Swing based desktop application** that calculates a student's academic performance using a modern **dashboard-style graphical user interface**.

This project allows users to enter subject names and marks, then automatically calculates **Total Marks, Average, Percentage, Grade, and Pass/Fail Status** with a visually appealing interface.

---

# 📌 Project Overview

The **Advanced Grade Calculator** is designed to simulate a **student result dashboard**. Unlike simple console programs, this application provides an **interactive graphical interface** built with **Java Swing**.

Users can enter:

- Student name
- Subject names
- Marks for multiple subjects

The system then calculates the student's academic performance and displays it with a **performance meter and result card**.

# 🧮 Grade Criteria

| Percentage | Grade |
|------------|------|
| 90% and above | A+ |
| 80% – 89% | A |
| 70% – 79% | B |
| 60% – 69% | C |
| 50% – 59% | D |
| Below 50% | F |

---

# 🎯 Pass Criteria

A student is considered **FAIL** if:

- Marks in **any subject are below 35**

Otherwise the student is considered **PASS**.
